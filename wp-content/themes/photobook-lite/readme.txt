== Description ==

Photobook is a modern and responsive photography WordPress theme for a photographer, event manager, wedding shoot and photography and fashion related websites. This theme is fully responsive and well coded with the latest CSS and HTML codes so it is a complete responsive for all the screen resolutions. Photobook theme has some sections that can display your latest photography services in a unique way. There has very modern layouts available such as services and slider so you can display your best shots in dynamic way. Photobook theme comes with some google fonts you can use the theme. Nice and easy slider can gives you some options to manage it as per you need. This theme is widget ready so you can use the widgets on the sidebar. This theme is testted with latest WordPress version and can be compatible with some popular plugins like wocommerce and contact form 7.
